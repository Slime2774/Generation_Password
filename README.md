# Generation Password 🔑

Простой и удобный консольный генератор безопасных паролей, написанный на языке **C#**. Проект создан в учебных целях для восстановления навыков программирования и практики работы с классами, обработкой ошибок и сторонними NuGet-пакетами.

---

## 🚀 Описание проекта

Приложение позволяет пользователю быстро сгенерировать случайный пароль заданной длины. В процессе работы можно выбрать, нужно ли добавлять в пароль специальные символы для повышения его надежности. Сгенерированный пароль автоматически копируется в буфер обмена.

### Что под капотом (C# фичи):
* **Разделение логики:** Основной цикл программы находится в `Program.cs`, а вся логика генерации инкапсулирована в классе `Password.cs`.
* **Автоматический буфер обмена:** Интеграция с библиотекой `TextCopy` (`ClipboardService`) для моментального копирования готового пароля.
* **Защита от дурака (Валидация):** Использование `int.TryParse()` предотвращает падение программы при вводе некорректных данных (например, букв вместо чисел).
* **Удобный UX:** Очистка консоли (`Console.Clear()`) перед каждым новым запросом и регистронезависимая проверка ответа (`StringComparison.OrdinalIgnoreCase`).
* **Оптимизация строк:** Для эффективной сборки пароля используется `StringBuilder`.

---

## 🛠️ Как это работает

1. Программа запрашивает желаемую **длину пароля** (с защитой от некорректного ввода).
2. Спрашивает, хотите ли вы использовать **специальные знаки** (`ДА` или `НЕТ` в любом регистре).
3. Генерирует случайный и безопасный пароль на основе выбранных параметров.
4. **Копирует результат в буфер обмена** — остаётся только вставить (`Ctrl+V`).
5. Позволяет повторить генерацию в чистом окне консоли.
