# Generation Password 🔑

Простой и удобный консольный генератор безопасных паролей, написанный на языке **C#**. Проект создан в учебных целях для закрепления базовых навыков работы с классами, строками и генерацией случайных чисел.

---

## 🚀 Описание проекта

Приложение позволяет пользователю быстро сгенерировать случайный пароль заданной длины. В процессе работы можно выбрать, нужно ли добавлять в пароль специальные символы для повышения его надежности.

### Что под капотом (C# фичи):
* **Разделение логики:** Основной цикл программы находится в `Program.cs`, а вся логика генерации инкапсулирована в классе `Password.cs`.
* **Оптимизация строк:** Для эффективной сборки пароля используется `StringBuilder`.
* **Гибкая настройка:** Динамическое объединение пулов символов в зависимости от выбора пользователя.

---

## 🛠️ Как это работает

1. Программа запрашивает желаемую **длину пароля**.
2. Затем спрашивает, хотите ли вы использовать **специальные знаки** (`ДА` или `НЕТ`).
3. Генерирует случайный и безопасный пароль на основе выбранных параметров.
4. Позволяет повторить генерацию в бесконечном цикле.
